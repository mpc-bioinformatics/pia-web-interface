PIA - Protein Inference Algorithms
Welcome to PIA

PIA is a toolbox for MS based protein inference and identification analysis.

PIA allows you to inspect the results of common proteomics spectrum identification search engines, combine them and conduct statistical analyses. The main focus of PIA lays on the integrated inference algorithms, i.e. concluding the proteins from a set of identified spectra. But it also allows you to inspect your peptide spectrum matches, calculate FDR values across different search engine results and visualize the correspondence between PSMs, peptides and proteins.

This web installation of PIA is mainly to get familiar with the way, PIA is used. The KNIME implementation is the most efficient way to use PIA in daily workflows. For developers, also the command line usage may be of interest. For more information on the last two, please also see PIA's GitHub page.
